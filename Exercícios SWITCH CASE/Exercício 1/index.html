Exercício 01

Questão

Usando a função Switch Case, desenvolva um programa que exiba um "menu" com as seguintes opções:

1- Reais para Dolares 2- Dolares para Reais 3- Reais para Libras 4- Libras para Reais 5- Dolares para Libras 6- Libras para Dolares 7- Finalizar Aplicação

Em seguida, solicitar um valor de conversão e exibir o resultado, de acordo com as moedas escolhidas, utilizando como referência os valores abaixo:

Libras = R$ 5,14		Dolar = R$ 4,44

A mensagem deve ser no padrão abaixo (exemplo):

O resultado do seu calculo: xxx Dolares Equivalem a xxx Reais.
